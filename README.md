# Renewable Energy ANN Project

This repository contains the files and deliverables for the renewable energy artificial neural network project.

## Deliverables

1. **Dataset & EDA Report**
   - Cleaned and raw datasets:
     - `solar_data_clean.csv`
     - `solar_data_raw.csv`
     - `weather_data_raw.csv`
   - EDA and correlation analysis results are represented by the generated figures in `figures/` and the summary in the final report.
   - Additional results data:
     - `results_table.csv`

2. **ANN Implementation Code**
   - Main notebook:
     - `ann_model_train.ipynb`
   - Saved model files:
     - `best_model.h5`
     - `best_model.keras`
   - Preprocessing scalers:
     - `scaler_X.pkl`
     - `scaler_y.pkl`

3. **Prediction Results**
   - Prediction output and performance metrics are available in:
     - `results_table.csv`
   - Visual results are also captured in `figures/`.

4. **Final Project Report**
   - PDF:
     - `deliverables/report.pdf`

5. **Presentation Slides**
   - Presentation file:
     - `deliverables/report_presentation.pptx`

## Notes

- The `deliverables/` folder contains the final report PDF and presentation slides.
- The `figures/` folder contains plots and visualizations used for analysis and results.
